На данный момент доступны руководства для следующих ОС:


  • Windows XP
  • Windows Vista
  • MacOS Panther 10.3
Нажимаем на яблоко и открываем "системные настройки", раздел "сети"
В верхней распускалке "месторасположение" выбираем "создать новое" В МакОси логика настроек сетей немного другая нежели в Windows или в популярных Линуксах. Суть - у вас для одного пользователя может существовать множество настроек сети (конфигураций - Locations). Вначале непривычно - но потом понимаешь, что это удобно. Для настольных машин это не очень актуально, а на ноутбуках - без этого не обойтись.
Вводим название
Заходим в панельку с выбором сетевых интерфейсов.
Отключаем ненужные (по желанию здесь можно переименовать сетевой интерфейс по вкусу)
Переключаемся в настройку сетевых параметров для выбранного сетевого интерфейса.
А здесь только выбираем DHCP и нажимаем кнопку применить.
Теперь создаем VPN соединение для выхода во внешнюю сеть. Переключившись в Finder идем в меню, переходим в паку с утилитами и находим иконку приложения "Internet Connect". Запускаем его...
Выбираем тип VPN-соединения (в нашем случае это PPTP) и нажимем кнопку продолжить...
Выбираем редактировать конфигурацию соединения...
И вводим данные нам при заключении контракта данные, не забыв указать имя сервера 192.168.100.1 Для сохранения нажимаем ОК и...
Зажигаем галочку показывать значок VPN.
